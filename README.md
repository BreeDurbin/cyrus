A simple initiative tracker for d&amp;d and other ttrpgs



Dependencies:
```
g++
make
cmake 3.28.3
qt6
boost libraries 1.83.0
```

To build:
```
# make build directory if doesn't already exist
mkdir build
cd build
# generate cmake files
cmake ..
# compile
make
```
